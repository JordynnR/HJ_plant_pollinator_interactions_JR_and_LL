# HJ_plant_pollinator_interactions_JR_-_LL
Final Project For BI410L (Data science in ecology).
what the project does, why is it useful, 
Our project investigates plant pollinator interactions with respect to native 
and non-native species, in high elevation meadows in the HJ Andrews research forest. 
This is a project for BI410L: Data Science for Ecology and Conservation at the 
University of Oregon Fall term 2025.
Collaborators are Jordynn Risdal and Lily Lawrence.
Included in this repository:
Project topic identification 
Literature review 
Proposed workflow 
HJ Andrews data set 
Data cleaning of raw data.
Data analyses 
Data visualizations 
Conclusion
